Tools

Integrated Development Environments

  • Pyzo - A starting IDE for Python
  • Eclipse - An open source community, whose projects are focused on development tools across the projects life cycle. (Java, C/C++)
  • Intellij IDEA - Community Edition is an open source version of the premium Intellij IDEA. This tool is great with Java, Groovy/Gradle, and others
  • PyCharm - Another Jet Brains (makers of Intellij) product, is the community edition of their professional Python IDE.

Text Editors

  • Vi/Vim/NeoVim - Command line based text editors
  • Sublime Text
  • Atom.io
