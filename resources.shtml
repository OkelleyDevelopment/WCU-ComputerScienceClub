The Great Page of Computer Science Resources

A little help in a complex field

WCU CS Club provides guidance to students but there are also fantastic resources across the web that explain subjects we cover in either better detail or more clarity (sometimes both!). The CS Club also welcomes peer-to-peer discussions so feel free to reach out to the club officers or advisors.

Tools

Integrated Development Enviroments

  • Pyzo - A starting IDE for Python
  • Eclipse - An open source community, whose projects are focused on developetment tools across the projects lifecycle. (Java, C/C++)
  • Intellij IDEA - Community Edition is an open source version of the premium Intellij IDEA. This tool is great with Java, Groovy/Gradle, and others
  • PyCharm - Another Jet Brains (makers of Intellij) product, is the community edition of their professional Python IDE.

Text Editors

  • Vi/Vim/NeoVim - Command line based text editors
  • Sublime Text
  • Atom.io
